# 🩺 Diabetes Prediction System

## Overview

This project uses Machine Learning algorithms to predict whether a person is diabetic based on medical attributes such as glucose level, blood pressure, insulin level, BMI, age, and other health indicators.

The project compares the performance of multiple machine learning models and uses the best-performing model for prediction.

---

## Algorithms Used

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

---

## Dataset Features

- Pregnancies
- Glucose
- BloodPressure
- SkinThickness
- Insulin
- BMI
- DiabetesPedigreeFunction
- Age
- Outcome

---

## Performance Metrics

The models are evaluated using:

- Accuracy Score
- Recall Score
- Precision Score

## Data Visualization

The project generates:

- Model Comparison Graphs
